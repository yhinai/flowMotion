## Production Notes

- Record at 1080p, 30fps
- Use screen capture with a clean browser window (no bookmarks bar, minimal tabs)
- Voiceover should be energetic but not rushed -- rehearse to hit timing marks
- If a live API call is slow during recording, have a pre-cached response ready as backup
- Total runtime target: 2:50-3:00 (leave a few seconds of buffer)
